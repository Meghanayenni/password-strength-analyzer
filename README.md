# Password Strength Analyzer
Intern ID: CITS5240
This is a cybersecurity mini-project that analyzes the strength of user-entered passwords in real time.

## Features

- Checks password length
- Checks uppercase and lowercase letters
- Checks numbers and special characters
- Shows password strength score
- Displays security checks
- Gives suggestions to improve weak passwords
- Shows entropy and estimated crack time
- Includes animated cyber-themed UI

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Download or clone this repository.
2. Open `index.html` in any web browser.

## Expected Outcome

The project helps users understand password security by analyzing password strength based on length, complexity, entropy, and character variety. It also gives suggestions to create stronger passwords.
